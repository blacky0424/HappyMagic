王子様や衛兵に見つからないようにしながら宝物庫を目指すステルスゲーム。
宝物庫にあるガラスの靴を破壊することでクリアとなる。
シンデレラは各コマンド（隠蔽、陽動、変装、聞き耳）を使用できる。
衛兵の状態はわからないため、聞き耳を使用し、各状態に適したコマンドを駆使して宝物庫を目指す。
時間制限、ライフが決められており、タイムアップもしくはライフ０でゲームオーバー。

遊び方:
左下のスティックを引っ張る方向に進む。
真ん中より奥に引っ張ると走行、内側だと歩行。
走行中は足音がでて、近くに衛兵、王子様がいるとばれてしまう可能性がある。
左上にあるハートの個数が残りライフ。
右上の時計が一周するとタイムアップ。時計をタッチするとメニューが開かれ一時停止。
ステージの様々な場所に各状態の衛兵が配置されている。

各プレイヤーコマンド
	隠蔽　カボチャをかぶって隠れる　スティック移動で解除。クールタイムあり。
	陽動　ネズミを放つ　　          タップした方向に進む。一定時間経過で消える。
　　　　変装　衛兵に変身　              一定時間で元の姿に戻る。
	聞き耳　衛兵の状態を知れる
衛兵・・・1人に注意力散漫、集中、巡回の各状態のどれかが割り当てられている。
	注意力散漫　変装、隠蔽を見抜けない
	集中　隠蔽のみ見抜けない
	巡回　隠蔽のみ見抜けない、ネズミにつられてしまう
王子様・・・隠蔽のみ見抜けない、見つかったら即ゲームオーバー（BadEnd）。

ステージの奥にある扉の前まで行くとガラスの靴を割るシーンに遷移。
ガラスの靴を手に取るムービーが流れる。（スマホのみ）
両手の指で左右の画面をタッチしながら、上から下に端末を振り下ろす。
スコアが表示されゲームクリア（TrueEnd）。
宝物庫のシーンはスマホのジャイロセンサーと加速度センサーを使用しているためスマホのみ動作。
